## 🌒 Концепт и сеттинг

> *После падения Алой Луны королевство Алшаполия раскололось на четыре фракции. Замки осыпались в туман, дороги поросли костями, а на месте бывших ярмарок открылись Чёрные Базары. Ты — герой, который собирает руины в новую корону. Или умирает, пытаясь.*

Игроки воплощают героев четырёх фракций, путешествуют по кольцу из 40 готических локаций (Усыпальница Королей, Башня Архимага, Гнилое Болото, Лунный Алтарь и так далее) и сражаются за владения с помощью карточных колод. Каждая клетка — мини-сценарий: торги, бой, квест или событие.

**Возраст:** 16+  
**Игроков:** 2–8  
**Длительность:** 60–120 минут  
**Формат:** физическая настолка + бесплатное цифровое компаньон-приложение (HUD, дайс-роллер, трекер хода)

## ⚙ Игровой цикл

Один ход — **шесть шагов**:

### 01 · БРОСОК
Бросьте `2d6`. Двигайте фишку по часовой стрелке. **Дубль** — повторите бросок. **Три дубля подряд** → в Тёмницу (пропуск 2 ходов).

### 02 · СОБЫТИЕ
Тип клетки определяет, что происходит:
- **Ничья собственность** → запускаются *Торги* (см. шаг 03)
- **Чужая собственность** → плати ренту по таблице владельца
- **Своя собственность** → бесплатно, можно построить страж-карту
- **Фортуна / Проклятие** → тяни карту события
- **Квест** → выбор и последствия из сюжетной деки
- **Тёмница / Святилище / Портал / Начало** — особые угловые клетки

### 03 · ТОРГИ МАСТИ *(преферанс-механика)*
Все игроки **втёмную** заявляют, сколько взяток возьмут (1–6) в TCG-партии за клетку. Высшая заявка становится **претендентом** — он играет бой со Стражем клетки.

> Если претендент не выполнит свою заявку — штраф **× 2 от цены клетки** в банк.

### 04 · TCG-БОЙ
Партия в духе Hearthstone:
- Колода 30 карт у каждой стороны
- Мана растёт 1 → 8 за ход
- На столе помещается до 7 существ с *ATK/HP*
- Заклинания одноразовые
- **Цель**: нанести 10 урона ИЛИ взять ровно заявленное число взяток

### 05 · ТРОФЕИ
- **Победа + заявка выполнена** → титул клетки твой, +XP, +золото
- **Победа, но недобрал/перебрал** → клетка твоя, но 0 XP
- **Поражение** → −HP, −золото в банк, заявка штрафуется ×2

### 06 · РОСТ
За каждый бой герой получает опыт. Уровень даёт:
- +1 в характеристику (Сила, Интеллект, Удача, Выносливость)
- +1 очко в дерево навыков класса
- На уровне 3 герой выбирает класс (Воин · Маг · Торговец · Некромант)
- Максимальный уровень — 20

## ⚔ Правила боя (TCG + торги)

### Колода
- Минимум **30 карт**, максимум **60**
- Не более **2 копий** одной карты обычной редкости, **1 копия** редкой и выше
- Класс определяет 50% колоды (фракционные карты), остальное — нейтрально

### Мана
- Старт: 1 мана-кристалл, растёт +1 за ход (до 8)
- Кристалл восстанавливается в начале каждого хода

### Существа
| Стат | Что значит |
|---|---|
| 🔵 **Мана** | Стоимость разыгрывания |
| 🟡 **Атака (ATK)** | Урон по существу или герою |
| 🔴 **Здоровье (HP)** | Сколько урона выдерживает |

Существо после появления имеет **сонную болезнь** — не может атаковать в тот же ход (исключения у Тени).

### Взятки *(преферанс-аналогия)*
Внутри партии есть концепт «раундов-взяток»:
- Каждый раунд оба игрока выкладывают по 1 карте на «стол взятки»
- Кто нанёс больше урона в раунде — берёт взятку
- Количество взяток сравнивается с заявкой по итогам партии
- Невзял свою заявку → ×2 штраф; перебрал → бой считается ничьей

### Победа в партии
Один из исходов:
1. **HP противника = 0** → мгновенная победа
2. **Выполнил заявку взяток** → победа с трофеями
3. **30 раундов прошло без исхода** → ничья (клетка остаётся ничьей до следующего хода)

## 🏆 Условия победы

Игра заканчивается, когда **один игрок** достигает любого из трёх условий:

| Путь | Требование |
|---|---|
| 👑 **ТРОН** | Собрать **3 полных региона** (все клетки одного цвета) |
| ⚔ **ГЕРОЙ** | Достичь **20 уровня** персонажа |
| 💎 **КОРОНА** | Завершить финальный квест **«Корона Алшаполии»** (4-актовая цепочка) |

Если игрок выбит из игры (банкрот + HP=0), его собственность уходит в банк и выставляется на повторные торги.

## 🏰 Фракции и классы

Все четыре фракции имеют свои колоды, портреты, цвета и стиль торгов.

### ⚜ Рыцари Зари *(класс: Воин)*
Лоялисты падшей короны. Танки, исцеление, налоговый бонус с союзных клеток. Любят прямые торги — берут много, но честно.

### 🩸 Орден Червя *(класс: Некромант)*
Культ погибели. Жертвоприношения, проклятия, призыв нежити из сброса. Их Стражи не умирают окончательно.

### 🪙 Вольное Купечество *(класс: Торговец)*
Гильдии и шпионы. Лучшие торги (−20% к ценам клеток), доступ к Чёрным Базарам, может перекупать заявки соперников.

### ✦ Высший Ковен *(класс: Маг)*
Архимаги. Сильные одноразовые заклинания, контроль колоды противника, телепорт по полю.

## 📱 Цифровое компаньон-приложение

Бесплатное приложение (доступ по QR-коду в коробке). Не обязательное — игра полностью играется без него — но сильно ускоряет учёт.

**Экраны:**
- **HUD героя** — портрет, HP/MN/XP бары, золото, текущая клетка, активный квест
- **Бой** — мини-TCG-стол: твоя рука, доска, враг, счётчик взяток (заявлено/взято/осталось)
- **Карта мира** — реальное положение всех 8 игроков, их собственность по цветам, рейтинг
- **Колода** — конструктор колоды между матчами
- **Кузня** — прокачка экипировки, дерево навыков

Эстетика приложения — отдельный handheld-девайс с пиксельным экраном и D-pad.

## 🚀 Локальный запуск

Проект — статический HTML, никакого билда не нужно.

### Вариант 1 — открыть напрямую
Двойной клик на `Алшаполия.html` в файловом менеджере. Браузер откроет канвас.

> ⚠ Некоторые браузеры (Chrome) блокируют локальные `fetch()` для боковых файлов состояния. Это не критично — канвас работает, просто состояние пана/зума не сохраняется между сессиями.

### Вариант 2 — локальный сервер (рекомендуется)
```bash
# Python 3
python -m http.server 8000

# Node
npx serve

# затем открыть в браузере:
# http://localhost:8000/Алшаполия.html
```

### Вариант 3 — GitHub Pages
1. В настройках репозитория: **Settings → Pages → Source → main → /(root)**
2. Через 1-2 минуты игра будет доступна по адресу `https://<твой-юзернейм>.github.io/<имя-репозитория>/Алшаполия.html`

## 🎨 Дизайн-система

### Палитра *(дефолтная)*

```
NIGHT      #0e0a1a   полночный фон
PANEL      #2a1f48   тёмный пурпур, основные плашки
RIM        #5b438f   рамки и обводки
GOLD       #e8b84c   валюта, легендарка, акценты
BLOOD      #c8313a   HP, кровь, тревога
BONE       #e8d9b0   текст
ARCANE     #4b88c2   мана, магия
HEX        #9558d6   некромантия, эпическая редкость
TOXIC      #66b542   яд, природа
```

### Типографика
- **Заголовки**: `Press Start 2P` (Google Fonts) — для угловых ярлыков, рамок, чипов
- **Дисплей**: `Jersey 15` — для больших готических заголовков
- **Body**: `Pixelify Sans` — для всего читабельного текста

### Принципы
- Все границы и тени — **2–3px чанки** (никаких размытых shadow-blur)
- Иллюстрации — намеренные SVG-силуэты как плейсхолдеры. Финальный арт = настоящий 16-bit pixel art (~64×48 спрайты)
- Скан-линии и дизеринг — на крупных панелях
- `image-rendering: pixelated` везде
